## MODIFIED Requirements

### Requirement: Mask terminology in Chinese translation
The system SHALL display "智能助手" instead of "面具" for all Mask-related UI text in the Simplified Chinese locale.

#### Scenario: Sidebar mask link displays new terminology
- **WHEN** user views the sidebar in Simplified Chinese
- **THEN** the mask link displays "智能助手" instead of "面具"

#### Scenario: Mask startup page title updated
- **WHEN** user opens the mask startup page
- **THEN** the title displays "智能助手启动页" instead of "面具启动页"

#### Scenario: Search placeholder updated
- **WHEN** user views the mask search interface
- **THEN** the placeholder displays "搜索智能助手" instead of "搜索角色面具"

#### Scenario: Share mask dialog updated
- **WHEN** user shares a mask
- **THEN** the dialog displays "分享此智能助手" instead of "分享此面具"

#### Scenario: Mask picker title updated
- **WHEN** user opens the mask picker
- **THEN** the title displays "挑选一个智能助手" instead of "挑选一个面具"
